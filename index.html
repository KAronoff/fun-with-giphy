Some information to help you get started...

  1. Fill in search topic and the number of gifs you want!
  2. When you press submit, two things happen:
    • You search results will appear on the right side of this page.
    • A button will appear in the previous search area. This button, when clicked, brings up the same topic and number of gifs
  3. To change the number of results from a previous search, type in the desired number in the Number of Results area, then click on one of your previous search buttons. The new number will be pinned to the button for future use. To change the number again, repeat the process.
  4. When you're ready to clear the buttons, press clear. This clears all but the 3 original buttons, and the gif area.
